QA automation test
This is a form for the QA Automation test.
Bonus point for using library or framework on top of Selenium Webdriver!
Bonus point for using Page object pattern.
Bonus point for "production ready" code.
Please use Java. Use Junit or TestNg, Maven or Gradle.

Test should pass in 3 browsers: Chrome, Firefox, Internet Explorer.

Steps:
1) Fill first and second question
2) Validate that third question is mandatory
3) Fill third question and go to another step
4) Fill next questions
5) Go back to first step
6) Reverse text in third question
7) Go to second step
8) Check that both questions are still filed
9) Go to last step
10) Fill last question and send form

Completed test upload to GitHub and send link via email to your contact person.
